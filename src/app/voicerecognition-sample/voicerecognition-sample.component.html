Voice Recognition

Usage samples

Voice Recognition is implemented as a directive. Therefore, it can extend any input or textarea to allow voice input whenever text is accepted.

To use it click on the mic icon and talk.

Language must be configured to improve recognition accuracy.

    
      <label for="searchField">Click the mic to talk:</label>
      <input id="searchField" luxVoiceRecognition language="es-ES">

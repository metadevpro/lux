Tooltip

Usage samples

Use it as a directive. Set tooltip text and placement.

Default Tooltip

    
  <button luxTooltip>Tooltip</button>
  
    
  <button luxTooltip content="Top" placement="top">Top</button>
  
      
  <button luxTooltip content="Bottom" placement="bottom">Bottom</button>
  
    
  <button luxTooltip content="Left" placement="left">Left</button>
  
    
  <button luxTooltip content="Right" placement="right">Right</button>

Tooltip from Component

    
  import { Component } from '@angular/core';
  @Component({
      template: `
        <span class="lux-tooltip" style="transition: opacity 200ms">Top Component</span>
      `
  })
  export class TooltipComponent {
      constructor()  { }
  }
  
    
  getComponentTooltip(): TooltipComponent {
    return TooltipComponent;
  }
  
    
  <button luxTooltip [content]="getComponentTooltip()" placement="top">Top Component</button>

Tooltip from TemplateRef

    
  <ng-template #contentTop>
    <span class="lux-tooltip" style="transition: opacity 200ms">Top TemplateRef</span>
  </ng-template>
  <button luxTooltip [content]="contentTop" placement="top">Top TemplateRef</button>

Documentation

Properties

  • luxTooltip:Directive

    Provides the tooltip to be shown.

  • content?: String | Component | TemplateRef

    Tooltip content. It can be a custom component, a custom TemplateRef or String. If it is a String, it display a default component and the content property is the message to show. If you want display a custom component or custom TemplateRef, the tooltip that you want display must have lux-tooltip class.

    Default: 'Tooltip'

  • placement?:String

    Tooltip position

    Values: Top, Bottom, Left or Right

    Default: Top

Styles

To customize look&fell change the lux-tooltip CSS class.

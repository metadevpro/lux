Lux Playground

Lux is a UI Component library for Angular created by Metadev.

Add @metadev/lux to the module: 

 
import {LuxModule} from '@metadev/lux';


  
@NgModule({

 imports: [
   LuxModule
 ]
 ...

Components

The following components are provided:

  • Breadcrumb
  • Checkbox
  • Filter
  • Input
  • Modal
  • Pagination
  • Radiogroup
  • Tooltip

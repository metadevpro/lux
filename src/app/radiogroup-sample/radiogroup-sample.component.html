Radiogroup

Usage samples

A radio group where user should pick one value.

Sample

    
  carBrands: RadioItem[] = [{
    name: 'radioHonda',
    label: 'Honda',
    value: 'HON'
  }, {
    label: 'Mazda',
    value: 'MZD'
  }, {
    label: 'Mitsubishi',
    value: 'MIT'
  }, {
    label: 'Lamborghini',
    value: 'LAM'
  }];
    
  
    
  <lux-radiogroup name="brand" [(ngModel)]="selectedValue" [items]="carBrands">
  </lux-radiogroup>

Selected value: {{ selectedValue }}

Disabled or Readonly

    
  <lux-radiogroup name="brand2" [items]="carBrands" [(ngModel)]="selectedValue2" [disabled]="true">
  </lux-radiogroup>

Selected value: {{ selectedValue2 }}

Documentation

The LuxRadioGroup component implements ControlValueAccessor, therefore binding using ngModel and name is supported.

Properties

  • itemsThe options for the radio group.
  • readonly If read-only is set the component is not editable.
  • disabled If disabled the component is not editable.
  • value Currrent value.

Events

  • valueChange Fires whenever the value changes.
  • itemsChange Fires whenever the input items changes.

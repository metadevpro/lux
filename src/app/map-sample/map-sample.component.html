Map

Attributes

mapId

        
<lux-map aria-label="MapId example" [mapId]="mapId"></lux-map>

center and zoom

        
<lux-map aria-label="Center example" [center]="geopoint" [zoom]="zoom"></lux-map>

markerPoint

        
<lux-map aria-label="MarkerPoint example" [markerPoint]="geopoint" [center]="geopoint" [zoom]="zoom"></lux-map>

readonly

        
<lux-map aria-label="Readonly example" [markerPoint]="geopoint" [center]="geopoint" [zoom]="zoom"[readonly]="true" ></lux-map>

Documentation

Properties

  • mapId id for the <div> containing the map. Must be unique. If left blank, it is autogenerated.
  • center The point the map is centered into.
  • zoom The zoom level of the map, with 0 representing the whole world and each level doubling the size. By default it's 16, which is enough to pinpoint an address.
  • markerPoint The point where a position marker is placed.
  • readonly If true, no position markers can be placed or moved through user interaction. If false, the user can click on the map to place a new marker or drag a marker to a new position.

Checkbox

Usage samples

Set checkbox label and change state with the properties value and disabled.

Checkbox with label enabled

    
<lux-checkbox name="closed" label="Is closed?">
</lux-checkbox>
<lux-checkbox name="hasCar" label="Has car?" [value]="true" [tabindex]="0">
</lux-checkbox>

<lux-checkbox name="closed" label="Is closed?">
</lux-checkbox>
<lux-checkbox name="hasCar" label="Has car?" [(value)]="true" [tabindex]="-1">
</lux-checkbox>

Checkbox disabled

    
<lux-checkbox name="closed" label="Is closed? [disabled]="true" [tabindex]="-1">
</lux-checkbox>
<lux-checkbox name="hasCar" label="Has car?" [disabled]="true" [(tabindex)]="-1" [value]="true">
</lux-checkbox>

<lux-checkbox name="closed" label="Is closed? [(disabled)]="true" [(tabindex)]="-1">
</lux-checkbox>
<lux-checkbox name="hasCar" label="Has car?" [(disabled)]="true" [(tabindex)]="-1" [(value)]="true">
</lux-checkbox>

Documentation

Properties

  • label Optional label for checkbox .
  • disabled If disabled the checkbox is not editable.
  • value Status of checkbox on or off.

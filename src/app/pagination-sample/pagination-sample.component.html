Documentation

Properties

  • PaginationInfo Data structure providing the pagination details.
    • total Total of items.
    • pageSize Number of items that it is show.
    • page Current page.
    • pagesToShow Pages to show in the toolbar.

Events

  • goToPage(n: number) Event triggered when click in one of pagination buttons. 'n' is the number of the page
  • reloadPage(n: number) Event triggered when changes the number of items to show. 'n' is the number of items to show

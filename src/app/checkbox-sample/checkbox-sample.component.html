Checkbox

Usage samples

Set checkbox label and change state with the properties value and disabled.

Checkbox with label enabled

    
<lux-checkbox name="closed" label="Is closed?"  [(ngModel)]="model.closed">
</lux-checkbox>

<lux-checkbox name="hasCar" label="Has car?" [(ngModel)]="model.hasCar">
</lux-checkbox>
Closed? Binded value: {{ model.closed }}
Has car? Binded value: {{ model.hasCar }}
{{ model | json }}

Checkbox disabled

    
<lux-checkbox name="closed" label="Is closed? [(disabled)]="true" [ngModel]="model.closed">
</lux-checkbox>

<lux-checkbox name="hasCar2" label="Has car? [disabled]="true" [ngModel]="model.hasCar">
</lux-checkbox>
Binded value: {{ model.closed }} Binded value: {{ model.hasCar }}

Documentation

The Checkbox component implements ControlValueAccessor, therefore binding using ngModel and name is supported.

Properties

  • inputId id for the input. If left blank, it is autogenerated.
  • label Optional label for checkbox .
  • disabled If disabled the checkbox is not editable.
  • value Status of checkbox on or off.

Events

  • valueChangeFired whenever the value changes.

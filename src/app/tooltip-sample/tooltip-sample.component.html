Tooltip

Usage samples

Use it as a directive. Set tooltip text, placement and delay.

Default Tooltip on top

    
  <button luxTooltip="Top" placement="top">Top</button>

Default Tooltip on bottom

      
  <button luxTooltip="Bottom" placement="bottom">Bottom</button>

Default Tooltip on the left

    
  <button luxTooltip="Left" placement="left">Left</button>

Default Tooltip on the right

    
  <button luxTooltip="Right" placement="right">Right</button>

Tooltip from Component on Top

    
  import { Component } from '@angular/core';
  @Component({
      template: `
        <span class="ng-tooltip" style="transition: opacity 200ms">Top Component</span>
      `
  })
  export class TooltipComponent {
      constructor()  { }
  }
  
    
  getComponentTooltip(): TooltipComponent {
    return TooltipComponent;
  }
  
    
  <button luxTooltip [content]="getComponentTooltip()" placement="top">Top Component</button>

Tooltip from TemplateRef on Top

    
  <ng-template #contentTop>
    <span class="ng-tooltip" style="transition: opacity 200ms">Top TemplateRef</span>
  </ng-template>
  <button luxTooltip [content]="contentTop" placement="top">Top TemplateRef</button>

Documentation

Properties

  • luxTooltip Provides the tooltip to be shown.
  • placement Placement allowed values are: top (default), bottom, left, and right.

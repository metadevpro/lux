Voice Recognition

Usage samples

VoiceRecongnition is implemented as a directive. Therefore it can extend any input or textarea to allow voice input whenever text is accepted.

To use it click on the mic icon and talk.

Laguage must be configured to improve recognition accuracy.

    
      <label for="searchField">Click to talk:</label>
      <input id="searchField" luxVoiceRecognition language="es-ES">

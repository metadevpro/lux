Lux Playground

Lux is a UI Component library for Angular created by Metadev.

Versions

  • For Angular 12.x use Lux version 0.6.x
  • For Angular 11.x use Lux version 0.5.6

How to install

To install, from the shell at your angular project root directory, type:

npm i --save @metadev/lux

Then, add the LuxModule from @metadev/lux to your application module: 

  
import { LuxModule } from '@metadev/lux';

@NgModule({
  declarations: [
    AppComponent
  ],
  imports: [
    BrowserModule,
    AppRoutingModule,
    LuxModule
  ],
  providers: [],
  bootstrap: [ AppComponent ]
})
export class AppModule { }

Components

The following components are provided:

  • Autocomplete
  • AutocompleteList
  • Breadcrumb
  • Checkbox
  • Filter
  • Geolocation
  • Input
  • Modal
  • Pagination
  • Radiogroup
  • Select
  • Tooltip
  • Voice Recognition

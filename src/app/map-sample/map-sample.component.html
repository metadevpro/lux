Map

Attributes

geopoint and zoom

        
        <lux-map aria-label="Geopoint example" [geopoint]="geopoint" [zoom]="zoom"></lux-map>

Documentation

Properties

  • geopoint The point the map is centered in.
  • zoom The zoom level of the map, with 0 representing the whole world and each level doubling the size. By default it's 16, which is enough to pinpoint an address.

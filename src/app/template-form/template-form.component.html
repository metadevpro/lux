Template Driven Form Use Case

Lux components can be used in Template Driven Angular Forms. Here is how:

A Template driven form binds directly with an model object. Each field uses double data-bind to:

  1. read and render the value when changed, and
  2. propagate changes to the model when the user edit the value

In this way, the model is mutable as the user is doing a direct edition of the content.

Technical Details

  • The directive ngModel is used to enable double data-binding.
  • Under the hood, Angular creates a FormControl per each field in the form.
  • Lux provides the corresponding formControlName properties when setting the property inputId.

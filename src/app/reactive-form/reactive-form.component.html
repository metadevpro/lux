Reactive Form Use Case

Lux components can be used in Reactive Angular Forms. Here is how:

On a Reactive Form, the binded object is consided immutable so any changes made to the model will trigger a new complete object model to be passed.

Single databing is used to pass along values to the controls. Changes are notified by events to give the application chances to trigger side-effect actions when needed.

Technical Details

  • Single data-bind is used to pass aling the model to the editors.
  • Model is considered immutable.

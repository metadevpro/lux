Documentation

Properties

  • inputId id for the input. If left blank, it is autogenerated.
  • aria-label Custom aria label forwarded to the geolocation.
  • step The discrete increment/decrement when using the arrows.
  • zoom The zoom level of the map, with 0 representing the whole world and each level doubling the size. By default it's 16, which is enough to pinpoint an address.
  • maxLatitude Maximum latitude.
  • minLatitude Minimum latitude.
  • maxLongitude Maximum longitude.
  • minLongitude Minimum longitude.

Events

  • valueChangeTriggered whenever the value changes. Pass the newValue as payload.
  • keyPressTriggered when a key is pressed. Pass the Keyboard event directly.

Documentation

Properties

  • inputId id for the inputs. The id of the latitude input will be this plus '$latitude' and the id for the longitude input will be this plus '$longitude'. If left blank, it is autogenerated.
  • aria-label Custom aria label forwarded to the datetime.
  • min Minimum date allowed.
  • max Maximum date allowed.
  • includeSeconds Whether seconds are included in the value.

Events

  • valueChangeTriggered whenever the value changes. Pass the newValue as payload.
  • keyPressTriggered when a key is pressed. Pass the Keyboard event directly.

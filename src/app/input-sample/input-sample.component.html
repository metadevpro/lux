Documentation

The LuxInput component implements ControlValueAccessor, therefore binding using ngModel and name is supported.

Properties

  • inputId id for the input. If left blank, it is autogenerated.
  • aria-label Custom aria label forwarded to the input.
  • step For numeric types: the discrete increment/decrement when using the arrows.
  • min For numeric types: maximum value.
  • max For numeric types: minimum value.

Events

  • valueChangeTriggered whenever the value changes. Pass the newValue as payload.
  • keyPressTriggered when a key is pressed. Pass the Keyboard event directly.

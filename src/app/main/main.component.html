Lux Playground

Lux is a UI Component library for Angular created by Metadev.

Add the LuxModule from @metadev/lux to your application module: 

  
import { LuxModule } from '@metadev/lux';

@NgModule({
  declarations: [
    AppComponent
  ],
  imports: [
    BrowserModule,
    AppRoutingModule,
    LuxModule
  ],
  providers: [],
  bootstrap: [ AppComponent ]
})
export class AppModule { }

Components

The following components are provided:

  • Autocomplete
  • AutocompleteList
  • Breadcrumb
  • Checkbox
  • Filter
  • Input
  • Modal
  • Pagination
  • Radiogroup
  • Select
  • Tooltip
  • Voice Recognition

Use Cases

  • Template Driven Forms
  • Reactive Forms

Tooltip

Usage samples

Use it as a directive. Set tooltip text, placement and delay.

Tooltip on top

<button luxTooltip="Tooltip on top" placement="top" delay="200">Top</button>

Tooltip on bottom

<button luxTooltip="Tooltip on bottom" placement="bottom" delay="200">Bottom</button>

Tooltip on the left

<button luxTooltip="Tooltip on the left" placement="left" delay="200">Left</button>

Tooltip on the right

<button luxTooltip="Tooltip on the rigth" placement="right" delay="200">Right</button>

Documentation

Properties

  • luxTooltip Provides the tooltip to be shown.
  • delay Animation delay to show and hide in milliseconds. Default to 200 ms.
  • placement Placement allowed values are: top (default), bottom, left, and right.
